Mac中安装使用nginx
2024-07-16

简单安装使用nginx的教程。

安装brew

  • 查看brew安装

    which brew

  • 安装brew

    /bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

  • 查看brew版本

    brew -v

安装nginx

  • 搜索nginx

    brew search nginx

  • 查看nginx信息

    brew info nginx

  • 安装成功查看版本

    nginx -v

配置nginx

  • 查看nginx安装信息

    brew info nginx

查看相关目录

  • 文档根目录

    /usr/local/var/www

  • 配置目录

    /usr/local/etc/nginx/nginx.conf

前端包部署

  • 进入部署目录

    cd /usr/local/var/www

  • 前端包放到该目录下

  • 进入配置目录

    /usr/local/etc/nginx/

  • 编辑nginx.conf文件

    修改监听端口+根目录html/test(其中test为前端包文件夹名)+增加url请求配置
