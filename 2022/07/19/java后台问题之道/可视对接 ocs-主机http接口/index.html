可视对接 ocs-子系统协议

1、接口说明

请求url： http://***/[action]

action： ***/reformer/interface/lifang

示例：http://192.168.30.233:8088/ocs/reformer/interface/lifang

请求参数

参数名 必选 类型 含义说明 备注说明
time 是 int unix时间戳 1571376463
data 是 string 加密后的数据 data为各请求参数的
加密内容，加密方式
如下

加密说明

加密方式为AES加密，秘钥为ASDyguFTDwXCkiNV ,偏移量为1234567890123456 .

2、HTTP接口

2.1 登录验证

接口地址：reformer/videophone/login

data参数说明

参数名 必选 类型 含义说明 备注说明
account 是 String 账号
password 是 String 密码
sn 是 String 设备序列号

响应示例

参数名 必选 类型 含义说明 备注说明
result 是 int 响应状态码
resultInfo 是 string 响应说明
responseData 否 Integer 管理员id

2.2 修改密码

接口地址：reformer/videophone/editPassword

data参数说明

参数名 必选 类型 含义说明 备注说明
adminId 是 Integer 账号id
passwordOld 是 String 旧密码
passwordNew 是 String 新密码
sn 是 String 设备序列号

响应示例

参数名 必选 类型 含义说明 备注说明
result 是 int 响应状态码
resultInfo 是 string 响应说明

2.3 获取通话记录

接口地址：reformer/videophone/queryCallRecord

data参数说明

参数名 必选 类型 含义说明 备注说明
sn 是 Strings 设备序列号
startTime 否 String 开始时间
endTime 否 String 结束时间

响应示例

参数名 必选 类型 含义说明 备注说明
result 是 int 响应状态码
resultInfo 是 string 响应说明
responseData 否 ArrayList

Object解析

参数名 必选 类型 含义说明 备注说明
callTime 是 Long 拨打时间
sn 是 String 设备序列号
faceId 是 Integer 人脸机id
faceIp 是 Stringr 人脸机Ip
faceName 是 String 人脸机名称
facePhone 是 String 人脸机分机号
StationIp 是 String 主机ip
StationPhone 是 String 主机号
talkTime 否 String HH:mm：ss
status 是 Integer 接听状态 0-未接通 1-接听 2-拒绝
adminId 是 Integer 管理员Id

响应示例
