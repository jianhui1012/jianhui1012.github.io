7.12 构建项目报lombok插件失效

  • 错误信息

使用lombok插件失效，编译报错，“You aren‘t using a compiler supported by lombok, so lombok will not work and has been disabled.”

  • 解决方法

    在以下位置加上该配置”-Djps.track.ap.dependencies=false”
