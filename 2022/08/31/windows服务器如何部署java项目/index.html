windows服务器如何部署java项目
2022-08-31

帮朋友搞个项目，服务器是windows的，数据库用的是mysql5.7+,第一次搞，正好记录下。

做什么

哪些东西需要放到服务器上

  • java项目打包

  • 导出创建数据库和相关表的sql文件

  • jdk+tomcat环境搭建

  • 启动tomcat和打包的服务

    • 打开tomcat服务器
      • sudo sh startup.sh
    • 关闭tomcat服务器
      • sudo sh shutdown.sh
    • 启动服务
      • java -jar jar包路径

  • 阿里云控制台增加入网的服务端口

  • 前端项目部署到服务器

ide项目打包发布到服务器

  • windows服务器安装配置ssh
  • ide安装阿里的一个上传插件,Alibaba cloud toolkit,方便后面ide打包直接发布到服务器
